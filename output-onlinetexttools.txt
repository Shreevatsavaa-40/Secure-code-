  Data flow — source to sink
  Tracks tainted values from a source (user input, request params, stdin) through variable
    assignments to a dangerous sink within this file. This is a lightweight, best-effort tracker — single file,
    top-to-bottom, no branches/loops modeled, no cross-function or cross-file tracking. It confirms real chains,
    it doesn't prove absence of others.
  



  OWASP Top 10:2021 coverage
  Every rule is tagged with a CWE and rolls up into an OWASP Top 10:2021 category by root cause, so a scan doubles as a rough OWASP coverage report. Counts below are live — computed from the rule set actually running above.
  



  CI/CD, Docker & CLI
  
    The browser tool above is a demo surface for the same engine — this is that engine, ported to Node with no dependencies, so it runs anywhere: a pre-commit hook, a Docker container, or a CI job that fails the build. cli.js is a direct port of the exact rule set, tokenizer, and scoring shown on this page, not a separate reimplementation.

    No dependencies — pure Node.js. Walks a file or directory, skips node_modules/.git/build output, and exits non-zero when a finding at or above --fail-on shows up.
    
      install & runCopy
      npm i -g .                                      # from the cli package dir — installs the `sentry-scan` command
sentry-scan .                                    # scan cwd, fail build on high/critical (default)
sentry-scan src/ --fail-on critical              # only fail on critical
sentry-scan . --format json --fail-on none       # machine-readable, never fail
sentry-scan . --format sarif -o results.sarif    # SARIF for GitHub/GitLab code scanning

  Where this fits
  This page is one piece of a broader rollout — same rule engine as the CI-integrated CLI/Docker tool, brought into a browser so results are shareable with anyone on the team, not just whoever has the terminal open.
  
    
      Phase 1 — live
      Workflow & deployment
      CLI + Docker image + GitHub Actions workflow, built and tested — see CI/CD & CLI above. Same rule engine as this page, ported to Node with zero dependencies. GitLab CI / Jenkins are a one-line swap of the run command.
    
    
      Phase 2 — this page
      Engine & accuracy
      91 rules across 15 categories, each with a CWE reference, an OWASP Top 10:2021 mapping, a fix-it hint, and vulnerable/fixed code side-by-side. A tokenizer reconstructs multi-line statements and strips comments before matching, and results roll up into a weighted 0–100 security score with a letter grade.
    
    
      Phase 3 — planned
      Platform & teams
      Auth, RBAC, scan history, and multi-project dashboards — turning this from a single-page tool into a shared service.
    
  

  What's next, in priority order
  Ranked by how much each closes the gap between "pattern matcher" and a tool teams actually trust in CI.
  
    
      Priority 1
      Real dataflow / taint tracking
      The biggest accuracy ceiling isn't more patterns — it's not knowing whether a variable actually traces back to user input. A lightweight taint pass (mark request/argv/env sources, follow assignments, flag when they reach a sink) would cut false positives more than any new rule.
    
    
      Priority 2
      Full-language AST via a real parser
      The current tokenizer strips comments and joins multi-line statements, which fixes the worst regex failure modes. A proper parser (e.g. a Python AST via a WASM build, or Babel's AST for JS) would let rules match on call structure and argument position instead of text shape — catching eval(x) passed through a wrapper function, for instance.
    
    
      Priority 3
      Suppression & baseline files
      A committed .sentryscan.yml allow-list for accepted-risk findings, so CI fails on new issues instead of every historical one on day one of adoption — the single biggest blocker to teams actually turning on "fail the build."
    
    
      Done
      SARIF export
      Shipped as part of Phase 1 — cli.js --format sarif uploads via github/codeql-action/upload-sarif, so findings show up as inline PR annotations instead of only a CLI log.
    
    
      Priority 5
      Semgrep/CodeQL rule pack import
      Let the engine consume existing community rule packs rather than hand-writing every pattern — trades some control for much faster coverage growth once the parser (Priority 2) exists to support it.
    
    
      Priority 6
      Auth, RBAC & scan history
      Same as Phase 3 above — valuable, but only after the engine itself (accuracy, suppression, CI visibility) is trustworthy enough that people want a shared history of it.

  // ---- Lightweight structural preprocessor (tokenizer) ----
  // Naive per-physical-line regex has two well-known failure modes: it fires on
  // matches sitting inside comments (false positives), and it misses matches
  // that span a wrapped multi-line statement, e.g.
  //   cursor.execute(
  //     f"SELECT * FROM users WHERE id = {user_id}"
  //   )
  // This pass walks the source once, strips // # /* */ comments, keeps string
  // contents intact (secrets live inside string literals), and joins any
  // statement that spans multiple physical lines — either via an open
  // (),[],{} depth or a trailing backslash continuation — into one logical
  // line before any rule pattern ever runs against it. It is a shallow,
  // token-aware pass rather than a full language parser, but it removes the
  // two biggest sources of regex-only noise.
  function buildLogicalLines(text) {
    const n = text.length;
    let i = 0;
    let line = 1;
    const logical = [];
    let buf = "";
    let bufStartLine = 1;
    let bufEndLine = 1;
    let depth = 0;

    function flush() {
      if (buf.trim().length) logical.push({ text: buf, startLine: bufStartLine, endLine: bufEndLine });
      buf = "";
    }

    while (i < n) {
      const ch = text[i];

      // line comments: // ... and # ...
      if ((ch === "/" && text[i + 1] === "/") || ch === "#") {
        while (i < n && text[i] !== "\n") i++;
        continue;
      }
      // block comments: /* ... */
      if (ch === "/" && text[i + 1] === "*") {
        i += 2;
        while (i < n && !(text[i] === "*" && text[i + 1] === "/")) { if (text[i] === "\n") line++; i++; }
        i = Math.min(i + 2, n);
        continue;
      }
      // python triple-quoted strings — kept, newlines flattened to spaces
      if ((ch === '"' && text.slice(i, i + 3) === '"""') || (ch === "'" && text.slice(i, i + 3) === "'''")) {
        const q = text.slice(i, i + 3);
        const start = i;
        i += 3;
        while (i < n && text.slice(i, i + 3) !== q) { if (text[i] === "\n") line++; i++; }
        i = Math.min(i + 3, n);
        buf += text.slice(start, i).replace(/\n/g, " ");
        continue;
      }
      // single/double/backtick-quoted strings — kept intact for secret matches
      if (ch === '"' || ch === "'" || ch === "`") {
        const q = ch;
        const start = i;
        i++;
        while (i < n && text[i] !== q) {
          if (text[i] === "\\") i++;
          else if (text[i] === "\n" && q !== "`") break;
          i++;
        }
        i = Math.min(i + 1, n);
        buf += text.slice(start, i).replace(/\n/g, " ");
        continue;
      }

      if (ch === "(" || ch === "[" || ch === "{") { depth++; buf += ch; i++; continue; }
      if (ch === ")" || ch === "]" || ch === "}") { depth = Math.max(0, depth - 1); buf += ch; i++; continue; }

      if (ch === "\\" && text[i + 1] === "\n") { buf += " "; i += 2; line++; bufEndLine = line; continue; }

      if (ch === "\n") {
        line++;
        if (depth > 0) { buf += " "; bufEndLine = line; i++; continue; }
        flush();
        i++;
        bufStartLine = line;
        bufEndLine = line;
        continue;
      }

      buf += ch;
      i++;
    }
    flush();
    return logical;
  }

  // Matches a single logical line against every rule. Shared by the
  // synchronous scanText() (kept for parity with the CLI/cli.js port, and
  // used for the taint tracer's own needs) and the chunked scanTextAsync()
  // the browser UI actually calls, so the two never drift apart.
  function matchLogicalLine(ll, nosecLines) {
    const out = [];
    for (let ln = ll.startLine; ln <= ll.endLine; ln++) {
      if (nosecLines.has(ln)) return out;
    }
    for (const rule of RULES) {
      rule.pattern.lastIndex = 0;
      if (rule.pattern.test(ll.text)) {
        out.push({
          ruleId: rule.id, category: rule.category, severity: rule.severity,
          confidence: rule.confidence, cwe: rule.cwe, owasp: owaspFor(rule.category),
          message: rule.message, line: ll.startLine, snippet: ll.text.trim().slice(0, 200),
          fixHint: rule.fixHint, example: rule.example,
        });
      }
    }
    return out;
  }

  function buildNosecLines(text) {
    const nosecLines = new Set();
    text.split(/\r?\n/).forEach((l, idx) => {
      if (/(\/\/|#)\s*nosec/i.test(l)) nosecLines.add(idx + 1);
    });
    return nosecLines;
  }

  function scanText(text) {
    const nosecLines = buildNosecLines(text);
    const findings = [];
    buildLogicalLines(text).forEach(ll => findings.push(...matchLogicalLine(ll, nosecLines)));
    return findings;
  }

  // Runtime-analysis pass used by the UI: identical matching logic to
  // scanText(), but chunked across setTimeout(0) ticks so a large paste
  // doesn't freeze the tab for the whole scan. Reports progress and honors
  // cancellation (either an explicit Cancel click, or a newer scan
  // superseding this one) via the token check.
  function scanTextAsync(text, onProgress, token, isCurrentToken) {
    return new Promise((resolve, reject) => {
      const nosecLines = buildNosecLines(text);
      const logicalLines = buildLogicalLines(text);
      const total = logicalLines.length;
      const findings = [];
      const CHUNK = 300; // logical lines per tick — 300 * 118 rules is comfortably under a frame budget
      let i = 0;

      function step() {
        if (!isCurrentToken(token)) { reject(new Error("cancelled")); return; }
        const end = Math.min(i + CHUNK, total);
        for (; i < end; i++) findings.push(...matchLogicalLine(logicalLines[i], nosecLines));
        if (onProgress) onProgress(i, total);
        if (i < total) setTimeout(step, 0);
        else resolve(findings);
      }
      setTimeout(step, 0);
    });
  }

  // ---- Scoring engine ----
  // Deliberately not a literal CVSS calculator (that needs a full attack
  // vector, so it's meaningless per-line) — this is a CVSS-inspired weighting
  // of severity against detection confidence, aggregated with diminishing
  // returns per rule so one noisy rule can't single-handedly tank the score.
  const SEVERITY_WEIGHT = { critical: 10, high: 6, medium: 3, low: 1 };
  const CONFIDENCE_WEIGHT = { high: 1, medium: 0.7, low: 0.4 };

  function findingRiskScore(f) {
    const raw = SEVERITY_WEIGHT[f.severity] * CONFIDENCE_WEIGHT[f.confidence];
    return Math.round(Math.min(10, raw) * 10) / 10;
  }

  function computeScore(findings) {
    const seenCount = {};
    const categoryLoss = {};
    let deduction = 0;

    findings.forEach(f => {
      seenCount[f.ruleId] = (seenCount[f.ruleId] || 0) + 1;
      const repeatFactor = 1 / Math.sqrt(seenCount[f.ruleId]); // 1st hit full weight, later hits taper off
      const base = SEVERITY_WEIGHT[f.severity] * CONFIDENCE_WEIGHT[f.confidence] * repeatFactor;
      deduction += base;
      categoryLoss[f.category] = (categoryLoss[f.category] || 0) + base;
    });

    const score = Math.max(0, Math.round(100 - deduction));
    let grade;
    if (score >= 90) grade = "A";
    else if (score >= 80) grade = "B";
    else if (score >= 65) grade = "C";
    else if (score >= 50) grade = "D";
    else grade = "F";

    const topCategories = Object.entries(categoryLoss)
      .sort((a, b) => b[1] - a[1])
      .slice(0, 4)
      .map(([category, loss]) => ({ category, loss: Math.round(loss * 10) / 10 }));

    return { score, grade, topCategories };
  }

  // ---- Taint tracking (source -> sink) ----
  // Real but intentionally scoped: a single linear pass over the file, tracking
  // which variables currently hold a value that originated from an untrusted
  // source. It follows plain assignment (`x = y`), a handful of C copy
  // functions (strcpy/strcat/memcpy), and scanf/gets/fgets writing into an
  // argument. It does NOT model branches, loops, function calls/returns
  // across functions, string concatenation, or object/array field access —
  // those need a real parser (AST) and call graph, which a regex pass can't
  // give you. What it reports is real: every flow below is a chain the tool
  // actually traced through this file, not a per-line guess.
  const TAINT_SOURCES = [
    { re: /\binput\s*\(/, label: "input()" },
    { re: /\b(request\.(GET|POST|args|form|json|data)|req\.(query|body|params))\b/, label: "HTTP request" },
    { re: /\bsys\.argv\b/, label: "argv" },
    { re: /\bprocess\.argv\b/, label: "argv" },
    { re: /\bprompt\s*\(/, label: "prompt()" },
    { re: /\b(document\.location|location\.search|location\.hash)\b/, label: "URL" },
    { re: /\bos\.environ\.get\s*\(/, label: "env var" },
    { re: /\bgetenv\s*\(/, label: "env var" },
  ];

  // Functions that write directly into a passed-in variable (C-style out-params),
  // handled separately from `x = y` assignment since the taint lands on an argument.
  const TAINT_ARG_WRITERS = [
    { re: /\b(scanf|fscanf)\s*\(([^)]*)\)/, label: "scanf() input" },
    { re: /\bgets\s*\(([^)]*)\)/, label: "gets() input", singleArg: true },
    { re: /\bfgets\s*\(([^)]*)\)/, label: "fgets() input" },
  ];

  const TAINT_SINKS = [
    { re: /\beval\s*\(/, label: "eval()", severity: "critical" },
    { re: /\bnew Function\s*\(/, label: "Function()", severity: "critical" },
    { re: /\bos\.system\s*\(/, label: "os.system()", severity: "critical" },
    { re: /(?<!\w)system\s*\(/, label: "system()", severity: "critical" },
    { re: /subprocess\.\w+\s*\([^)]*shell\s*=\s*True/, label: "subprocess(shell=True)", severity: "critical" },
    { re: /child_process\.(exec|execSync)\s*\(/, label: "child_process.exec()", severity: "critical" },
    { re: /\.execute\s*\(/, label: "SQL execute()", severity: "high" },
    { re: /\.innerHTML\s*=/, label: "innerHTML assignment", severity: "high" },
    { re: /\bprintf\s*\(/, label: "printf()", severity: "high" },
    { re: /\bstrcpy\s*\(/, label: "strcpy()", severity: "high" },
    { re: /\bstrcat\s*\(/, label: "strcat()", severity: "high" },
    { re: /\bpickle\.(load|loads)\s*\(/, label: "pickle.load()", severity: "high" },
    { re: /yaml\.load\s*\(/, label: "yaml.load()", severity: "high" },
    { re: /\bopen\s*\(/, label: "open()", severity: "medium" },
  ];

  function argNames(argsStr) {
    return argsStr.split(",").map(s => s.trim())
      .filter(s => !/^["'].*["']$/.test(s))          // skip string literals (format strings)
      .filter(s => !/^(stdin|stdout|stderr)$/.test(s))
      .map(s => s.replace(/^&/, "").replace(/\[.*\]$/, "").trim())
      .filter(s => /^[A-Za-z_]\w*$/.test(s));
  }

  function wordIn(name, text) {
    return new RegExp("\\b" + name.replace(/[.*+?^${}()|[\]\\]/g, "\\$&") + "\\b").test(text);
  }

  const FIRST_ARG_ONLY = new Set(["printf()", "SQL execute()"]);

  function firstArgText(line, openParenIdx) {
    let depth = 0, out = "";
    for (let i = openParenIdx; i < line.length; i++) {
      const c = line[i];
      if (c === "(" || c === "[" || c === "{") { depth++; out += c; continue; }
      if (c === ")" || c === "]" || c === "}") { if (depth === 0) break; depth--; out += c; continue; }
      if (c === "," && depth === 0) break;
      out += c;
    }
    return out;
  }

  function sinkMatchesVar(sink, line, varName) {
    if (!FIRST_ARG_ONLY.has(sink.label)) return wordIn(varName, line);
    const m = sink.re.exec(line);
    if (!m) return false;
    const openParen = line.indexOf("(", m.index);
    if (openParen === -1) return false;
    return wordIn(varName, firstArgText(line, openParen + 1));
  }

  function traceTaint(text) {
    const lines = text.split(/\r?\n/);
    const taint = {}; // varName -> { label, originLine, path: [{var, line, snippet}] }
    const flows = [];
    const seen = new Set();

    lines.forEach((rawLine, idx) => {
      const line = rawLine;
      const lineNo = idx + 1;
      if (/(\/\/|#)\s*nosec/i.test(line)) return;

      // 1. Sink check, using taint state as of BEFORE this line's own updates.
      for (const sink of TAINT_SINKS) {
        sink.re.lastIndex = 0;
        if (!sink.re.test(line)) continue;
        for (const varName in taint) {
          if (!sinkMatchesVar(sink, line, varName)) continue;
          const key = varName + "|" + taint[varName].originLine + "|" + lineNo + "|" + sink.label;
          if (seen.has(key)) continue;
          seen.add(key);
          flows.push({
            varName, sourceLabel: taint[varName].label, sourceLine: taint[varName].originLine,
            path: taint[varName].path.slice(), sinkLabel: sink.label, sinkLine: lineNo,
            sinkSnippet: line.trim().slice(0, 160), severity: sink.severity,
          });
        }
      }

      // 2. scanf/gets/fgets writing directly into an argument.
      for (const w of TAINT_ARG_WRITERS) {
        const m = line.match(w.re);
        if (!m) continue;
        const names = w.singleArg ? argNames(m[1] || "") : argNames(m[2] || m[1] || "");
        names.forEach(v => {
          taint[v] = { label: w.label, originLine: lineNo, path: [{ var: v, line: lineNo, snippet: line.trim().slice(0, 120) }] };
        });
      }

      // 3. C copy functions: dest inherits src's taint.
      const copyMatch = line.match(/\b(strcpy|strcat|memcpy)\s*\(\s*([A-Za-z_]\w*)\s*,\s*([A-Za-z_]\w*)/);
      if (copyMatch) {
        const dest = copyMatch[2], src = copyMatch[3];
        if (taint[src]) {
          taint[dest] = { label: taint[src].label, originLine: taint[src].originLine,
            path: [...taint[src].path, { var: dest, line: lineNo, snippet: line.trim().slice(0, 120) }] };
        }
      }

      // 4. Plain assignment: x = <expr>  (excludes ==, <=, >=, != via the lookahead/prior-char check).
      const assign = line.match(/^\s*(?:[A-Za-z_][\w:<>*&]*\s+)?([A-Za-z_]\w*)\s*=(?!=)\s*(.+?);?\s*$/);
      if (assign) {
        const varName = assign[1], rhs = assign[2];
        const srcHit = TAINT_SOURCES.find(s => s.re.test(rhs));
        if (srcHit) {
          taint[varName] = { label: srcHit.label, originLine: lineNo, path: [{ var: varName, line: lineNo, snippet: line.trim().slice(0, 120) }] };
        } else {
          const propagatedFrom = Object.keys(taint).find(v => v !== varName && wordIn(v, rhs));
          if (propagatedFrom) {
            const prior = taint[propagatedFrom];
            taint[varName] = { label: prior.label, originLine: prior.originLine,
              path: [...prior.path, { var: varName, line: lineNo, snippet: line.trim().slice(0, 120) }] };
          } else if (taint[varName]) {
            delete taint[varName]; // reassigned to a clean value — taint cleared
          }
        }
      }
    });

    return flows;
  }

  function escapeSvg(str) {
    return String(str).replace(/[&<>"']/g, ch => ({ "&": "&amp;", "<": "&lt;", ">": "&gt;", '"': "&quot;", "'": "&#39;" }[ch]));
  }

  function renderFlowGraphSVG(flow) {
    const steps = flow.path.length ? flow.path : [{ var: flow.varName, line: flow.sourceLine }];
    const nodes = [
      { label: flow.sourceLabel, sub: "line " + flow.sourceLine, type: "source" },
      ...steps.slice(1).map(p => ({ label: p.var, sub: "line " + p.line, type: "step" })),
      { label: flow.sinkLabel, sub: "line " + flow.sinkLine, type: "sink" },
    ];
    const boxW = 148, boxH = 46, gap = 44, startX = 16, y = 26;
    const totalW = nodes.length * boxW + (nodes.length - 1) * gap + 32;
    let svg = `<svg viewBox="0 0 ${totalW} 96" xmlns="http://www.w3.org/2000/svg" style="width:100%;height:auto;display:block;">`;
    svg += `<defs><marker id="arrow-${flow.sinkLine}-${flow.varName.replace(/\W/g,'')}" viewBox="0 0 10 10" refX="9" refY="5" markerWidth="7" markerHeight="7" orient="auto-start-reverse"><path d="M0,0 L10,5 L0,10 z" fill="#576273"/></marker></defs>`;
    nodes.forEach((n, i) => {
      const x = startX + i * (boxW + gap);
      const fill = n.type === "source" ? "#173B37" : n.type === "sink" ? "rgba(244,63,94,0.15)" : "#19222C";
      const stroke = n.type === "source" ? "#5EEAD4" : n.type === "sink" ? "#F43F5E" : "#232E3A";
      const textColor = n.type === "sink" ? "#F43F5E" : n.type === "source" ? "#5EEAD4" : "#E6EDF3";
      svg += `<rect x="${x}" y="${y}" width="${boxW}" height="${boxH}" rx="8" fill="${fill}" stroke="${stroke}" stroke-width="1.5"/>`;
      svg += `<text x="${x + boxW / 2}" y="${y + 19}" text-anchor="middle" font-family="IBM Plex Mono, monospace" font-size="11" fill="${textColor}" font-weight="600">${escapeSvg(String(n.label).slice(0, 20))}</text>`;
      svg += `<text x="${x + boxW / 2}" y="${y + 34}" text-anchor="middle" font-family="IBM Plex Mono, monospace" font-size="9.5" fill="#576273">${escapeSvg(n.sub)}</text>`;
      if (i < nodes.length - 1) {
        const x1 = x + boxW, x2 = x + boxW + gap - 4;
        svg += `<line x1="${x1}" y1="${y + boxH / 2}" x2="${x2}" y2="${y + boxH / 2}" stroke="#576273" stroke-width="1.5" marker-end="url(#arrow-${flow.sinkLine}-${flow.varName.replace(/\W/g,'')})"/>`;
      }
    });
    svg += `</svg>`;
    return svg;
  }

  const MAX_FILE_BYTES_HARD = 25 * 1024 * 1024; // hard cap — refuse to even attempt reading it

  function formatBytes(n) {
    if (n < 1024) return `${n} B`;
    if (n < 1024 * 1024) return `${(n / 1024).toFixed(1)} KB`;
    return `${(n / (1024 * 1024)).toFixed(2)} MB`;
  }

  // Crude but cheap: a NUL byte or a high ratio of control characters in the
  // first couple KB means this is very likely not source code (an image,
  // archive, etc. dropped by mistake), worth a warning rather than a silent
  // wall of regex noise.
  function looksBinary(sample) {
    if (!sample.length) return false;
    let suspicious = 0;
    for (let i = 0; i < sample.length; i++) {
      const code = sample.charCodeAt(i);
      if (code === 0) return true;
      if (code < 9 || (code > 13 && code < 32)) suspicious++;
    }
    return suspicious / sample.length > 0.05;
  }

  function loadFile(file) {
    if (!file) return;
    fileMeta.className = "file-meta";

    if (file.size > MAX_FILE_BYTES_HARD) {
      fileMeta.className = "file-meta error";
      fileMeta.textContent = `${file.name} is ${formatBytes(file.size)} — too large for the in-browser scanner (limit ${formatBytes(MAX_FILE_BYTES_HARD)}). Use the CLI for files or repos this size.`;
      return;
    }

    fileMeta.textContent = `Reading ${file.name}...`;
    const reader = new FileReader();
    const willTruncate = file.size > MAX_FILE_BYTES;

    reader.onprogress = (ev) => {
      if (ev.lengthComputable && file.size > 1024 * 1024) {
        fileMeta.textContent = `Reading ${file.name}... ${formatBytes(ev.loaded)} / ${formatBytes(file.size)}`;
      }
    };

    reader.onerror = () => {
      fileMeta.className = "file-meta error";
      fileMeta.textContent = `Couldn't read ${file.name} — try again, or open it in an editor and paste the contents instead.`;
    };

    reader.onload = (ev) => {
      let text = ev.target.result;
      if (looksBinary(text.slice(0, 2000))) {
        fileMeta.className = "file-meta warn";
        fileMeta.textContent = `${file.name} looks like a binary file, not source code — loaded anyway, but expect noisy/meaningless results.`;
      } else if (willTruncate) {
        text = text.slice(0, MAX_FILE_BYTES);
        fileMeta.className = "file-meta warn";
        fileMeta.textContent = `${file.name} is ${formatBytes(file.size)} — truncated to the first ${formatBytes(MAX_FILE_BYTES)} for the in-browser scanner. For the full file, use the CLI.`;
      } else {
        fileMeta.textContent = `${file.name} (${formatBytes(file.size)})`;
      }
      codeInput.value = text;
      updateLineCount();
    };

    reader.readAsText(file);
  }

  function escapeHtml(str) {
    return str.replace(/[&<>"']/g, ch => ({
      "&": "&amp;", "<": "&lt;", ">": "&gt;", '"': "&quot;", "'": "&#39;"
    }[ch]));
  }

  function renderResults(findings, failOn) {

      .join("");

    const sorted = [...findings].sort((a, b) => SEVERITY_ORDER[b.severity] - SEVERITY_ORDER[a.severity] || a.line - b.line);

    resultsBody.innerHTML = sorted.map((f, i) => `

        <div class="finding-top">
          <span class="sev-tag ${f.severity}">${f.severity}</span>

        </div>` : ""}

    `).join("");
  }

  function renderScore(findings) {
    const { score, grade, topCategories } = computeScore(findings);
    const gradeColor = { A: "var(--safe)", B: "var(--accent)", C: "var(--medium)", D: "var(--high)", F: "var(--critical)" }[grade];

    scoreRing.style.setProperty("--ring-pct", score);
    scoreRing.style.setProperty("--ring-color", gradeColor);
    scoreNum.textContent = score;
    scoreGrade.textContent = `Grade ${grade} \u00b7 Security score ${score}/100`;
    scoreGrade.style.color = gradeColor;
    scoreCats.innerHTML = topCategories.length
      ? topCategories.map(c => `<span class="score-cat-chip">${CATEGORY_LABEL[c.category] || c.category} \u2212${c.loss}</span>`).join("")
      : `<span class="score-cat-chip">No deductions</span>`;
    scoreBox.style.display = "flex";
  }

  let cancelRequested = false; // set by the Cancel button, scoped to the current token

  scanBtn.addEventListener("click", async () => {

    if (!text.trim()) {
      resultsBody.innerHTML = `<div class="empty-state"><div>&#9679;&#9675;&#9675;</div><div>Nothing to scan yet.<br>Paste some code first.</div></div>`;
      summaryStrip.style.display = "none";
      scoreBox.style.display = "none";
      document.getElementById("dataflowSection").style.display = "none";
      return;
    }

    const myToken = ++scanToken;
    cancelRequested = false;
    scanBtn.disabled = true;
    cancelScanBtn.style.display = "inline-block";
    scanStatus.textContent = "Scanning...";
    sweepLayer.classList.add("active");
    resultsBody.innerHTML = "";
    summaryStrip.style.display = "none";
    scoreBox.style.display = "none";
    document.getElementById("dataflowSection").style.display = "none";
    scanProgress.style.display = "block";
    scanProgressBar.style.width = "0%";

    const startedAt = performance.now();

    try {
      const findings = await scanTextAsync(
        text,
        (done, total) => {
          scanProgressBar.style.width = `${Math.round((done / Math.max(total, 1)) * 100)}%`;
          scanStatus.textContent = `Scanning... ${done.toLocaleString()} / ${total.toLocaleString()} lines`;
        },
        myToken,
        (t) => t === scanToken && !cancelRequested
      );

      if (myToken !== scanToken) return; // superseded by a newer scan — let that one own the UI

      const elapsedMs = Math.round(performance.now() - startedAt);
      const failOn = failOnSelect.value;
      renderResults(findings, failOn);
      renderScore(findings);
      renderDataFlow(traceTaint(text));
      resetScanUi();

      const lineTotal = text.split(/\r?\n/).length;
      const rate = elapsedMs > 0 ? Math.round((lineTotal / elapsedMs) * 1000) : lineTotal;
      const timing = `${elapsedMs < 1000 ? elapsedMs + "ms" : (elapsedMs / 1000).toFixed(1) + "s"} \u00b7 ~${rate.toLocaleString()} lines/s`;

      if (failOn === "none") {
        scanStatus.textContent = `${findings.length} finding(s) \u00b7 ${timing}`;
      } else {
        const threshold = SEVERITY_ORDER[failOn];
        const blocking = findings.filter(f => SEVERITY_ORDER[f.severity] >= threshold).length;
        scanStatus.textContent = `${blocking > 0 ? `${blocking} blocking / ${findings.length} total` : `${findings.length} finding(s), none blocking`} \u00b7 ${timing}`;
      }
    } catch (err) {
      if (myToken !== scanToken) return; // a newer scan already started; it owns the UI now
      resetScanUi();
      scanStatus.textContent = "Scan cancelled";
    }
  });
